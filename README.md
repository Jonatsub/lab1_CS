# Lab 1 – Centraliserad säkerhetsövervakning

Detta repo används för Lab 1 i kursen Nätverks-, OT- och AI-säkerhet.

## Syfte
Att bygga en enkel labbmiljö för centraliserad säkerhetsövervakning med stöd för AI-baserad anomalidetektion.

## Struktur
- `docker-compose.yml` – Wazuh-stack
- `configs/` – konfigurationsfiler
- `scripts/` – hjälpskript
- `screenshots/` – bilder till dokumentation
- `docs/` – anteckningar och arkitektur
- `src/` – Python-kod för test och AI-del
